## ACE-Step整合包使用说明
首先将网盘内的软件压缩包下载到本地电脑上并解压，然后双击启动软件.exe。

高配英伟达显卡选正常模式，低配英伟达显卡选低显存模式。

软件成功启动后会自动打开webui界面。

视频教程及效果演示：https://www.youtube.com/watch?v=PLBoVQklfhk

## 注意事项
英伟达显卡显存不低于6G

支持英伟达50系列显卡

英伟达20系列或是英伟达显卡显存较低(如12G以下)电脑建议使用低显存模式

使用前请将英伟达显卡驱动更新到最新版本

只支持Windows 10或11

软件运行路径中不要有非英文字符和空格，待处理文件素材也要注意
